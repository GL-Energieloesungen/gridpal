# Gridpal Vendor Binding

Manages and serves information about Vendors in the database.


## Dependencies

- `org.json`
- `sqlite-jdbc`


## REST API

BASE = `vendors`

### Endpoints

- `/interface`
- `/scan`
- `/manufacturer`
- `/device`
- `/device/config`
- `/registermap`

_Note: The endpoints are relative to `$BASE`._
